# AI Chatbot Demo - 고객 시연 가이드

> VIP 고객 시연을 위한 단계별 안내서

---

## 사전 준비

### 1. 서버 실행

```bash
# 환경 변수 설정
export OPENAI_API_KEY=your-api-key

# 서버 실행
./gradlew bootRun
```

### 2. 접속 URL 확인

| URL | 용도 |
|-----|------|
| http://localhost:8080 | 데모 페이지 |
| http://localhost:8080/swagger-ui.html | API 문서 |
| http://localhost:8080/h2-console | DB 콘솔 |

---

## 시연 시나리오

### 📍 Scene 1: 데모 페이지 소개 (2분)

1. http://localhost:8080 접속
2. **설명 포인트:**
   - "API를 직접 호출하지 않아도 바로 AI와 대화할 수 있습니다"
   - "회원가입/로그인 후 즉시 사용 가능합니다"

### 📍 Scene 2: 회원가입 & 로그인 (1분)

1. **회원가입** 탭 클릭
2. 정보 입력:
   - 이름: `홍길동`
   - 이메일: `demo@example.com`
   - 비밀번호: `demo1234`
3. **로그인** 후 채팅 화면 전환

**설명 포인트:**
- "JWT 토큰 기반 인증으로 보안이 유지됩니다"

### 📍 Scene 3: AI 대화 시연 (3분)

#### 기본 대화
```
👤 "안녕하세요, 자기소개 해주세요."
🤖 (실시간 스트리밍 응답)
```

**설명 포인트:**
- "답변이 실시간으로 타이핑되는 것처럼 보이죠? SSE 스트리밍 기술입니다"

#### 컨텍스트 유지 시연
```
👤 "Kotlin이 뭔가요?"
🤖 (Kotlin 설명)

👤 "그럼 Java와 뭐가 다른가요?"
🤖 (이전 대화를 기억하고 비교 답변)
```

**설명 포인트:**
- "30분 이내의 대화는 맥락이 유지됩니다"
- "같은 스레드 내에서 이전 대화를 기억합니다"

#### 복잡한 질문
```
👤 "1부터 10까지 천천히 세어주세요."
🤖 1... 2... 3... (스트리밍 효과 극대화)
```

### 📍 Scene 4: Swagger UI 시연 (2분)

1. http://localhost:8080/swagger-ui.html 접속
2. **설명 포인트:**
   - "모든 API가 문서화되어 있습니다"
   - "개발자가 바로 테스트할 수 있습니다"

3. API 호출 시연:
   - `POST /api/auth/login` → 토큰 발급
   - `Authorize` 버튼으로 토큰 설정
   - `POST /api/chats` → 대화 생성

### 📍 Scene 5: 확장 로드맵 설명 (2분)

`EXTENSION_ROADMAP.md` 기반 설명:

1. **현재 상태 (MVP)**
   - OpenAI API 직접 연동
   - 스레드 기반 대화 관리
   - 실시간 스트리밍

2. **Phase 1: RAG 도입**
   - "대외비 문서를 학습시켜 도메인 특화 챗봇으로 확장 가능합니다"
   - "예: 사내 규정, 제품 매뉴얼, FAQ 자동 답변"

3. **예상 일정**
   - RAG 기본 구현: 4-6주
   - 전체 고도화: 2-3개월

## FAQ 대응

### Q: "응답 속도가 느린 것 같은데요?"
> "OpenAI API 응답 시간에 의존합니다. 프로덕션에서는 캐싱, 모델 최적화로 개선 가능합니다."

### Q: "한국어 품질은 어떤가요?"
> "GPT-4 기반으로 한국어도 자연스럽게 처리됩니다. 필요시 Fine-tuning으로 품질 향상 가능합니다."

### Q: "보안은 어떻게 되나요?"
> "JWT 토큰 인증, HTTPS 적용, API 키 환경변수 관리. 프로덕션에서는 Azure OpenAI로 데이터 주권 확보 가능합니다."

### Q: "비용은 얼마나 드나요?"
> "GPT-4o-mini 기준 월 1만 대화 시 약 $10-30. RAG 도입 시 $100-200 수준입니다."

### Q: "자사 문서 학습은 언제 가능한가요?"
> "RAG 파이프라인 구축에 4-6주 예상됩니다. EXTENSION_ROADMAP.md에 상세 계획이 있습니다."

## 트러블슈팅

### 서버가 안 켜질 때
```bash
# 포트 확인
lsof -i :8080

# 강제 종료
kill -9 <PID>
```

### OpenAI API 오류
- API 키 확인: `echo $OPENAI_API_KEY`
- 잔액 확인: https://platform.openai.com/usage

### 데이터베이스 초기화
- H2 콘솔 접속 → `DROP ALL OBJECTS`
- 서버 재시작
